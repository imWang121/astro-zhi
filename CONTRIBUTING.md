# Contributing

感谢你考虑为 zhi 做贡献！

## 如何贡献

### 报告问题

如果你发现了 bug 或有功能建议，请[创建 Issue](https://github.com/imWang121/astro-zhi/issues)，并包含：

- 问题的详细描述
- 复现步骤
- 预期行为
- 实际行为
- 截图（如果适用）

### 提交代码

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 代码规范

- 使用 TypeScript
- 遵循现有代码风格
- 添加必要的注释
- 确保构建通过 (`npm run build`)

### 开发流程

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```

## 许可证

通过贡献代码，你同意你的代码将以 MIT 许可证授权。
